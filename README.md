# Comparative Analysis of EV Charging Methods (AC vs DC Fast Charging)

## Project Overview

This project presents a comparative analysis of Electric Vehicle (EV) charging methods, focusing on **AC Charging** and **DC Fast Charging**. The study evaluates charging speed, efficiency, battery performance, infrastructure requirements, installation costs, and practical applications.

A responsive website has been developed using HTML, CSS, JavaScript, Bootstrap, and Chart.js to present the project findings in an interactive and user-friendly manner.

---

## Objectives

- Study the working principles of AC and DC charging systems.
- Compare charging time and efficiency.
- Analyze battery State of Charge (SOC) characteristics.
- Evaluate battery degradation effects.
- Compare installation and infrastructure costs.
- Identify suitable applications for each charging method.
- Present simulation results using MATLAB and Simulink.
